# MERN Bookstore

A compact full-stack MERN project with:
- Express + MongoDB backend
- React + Vite frontend
- CRUD operations for books

## Project structure
- backend/ - Node/Express API
- frontend/ - React/Vite app

## Setup
1. Install dependencies:
   ```bash
   npm run install:all
   ```
2. Start both servers:
   ```bash
   npm run dev
   ```
3. Open the frontend at http://localhost:3000

## Environment
The backend uses the MongoDB connection string from backend/.env.
